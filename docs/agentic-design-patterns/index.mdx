# Agentic Design Patterns

Following is an outline of the agentic design patterns which we cover alongside their implementation in LangGraph. Each pattern is designed to address specific challenges in building agentic systems, and they often come with their own set of tradeoffs. Understanding these patterns can help you choose the right approach for your specific use case.
